# Asset Licenses

The repository is prepared for public release with no third-party food photo
assets bundled.

## Included Assets

- `public/favicon.svg` is an original simple SVG icon created for this project
  and is licensed under the repository's MIT license.
- Food cards render Unicode emoji/text labels instead of bundled photographs.

## Removed Risk Area

Previously bundled food photos and generic placeholder images did not include
source or license records. They were removed to avoid publishing assets whose
copyright status cannot be verified.

## Adding Assets Later

When adding images, audio, fonts, icons, datasets, or other non-code assets,
include all of the following in this file:

- Source URL or creator.
- License name and link.
- Required attribution text.
- Any usage restrictions, including commercial-use limits.
