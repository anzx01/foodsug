# Qwen VL API Notes

This project calls Alibaba Cloud DashScope through an OpenAI-compatible chat
completions endpoint. To avoid copying vendor documentation into this
repository, keep only project-specific integration notes here and refer to the
official documentation for complete and current API details.

## Project Configuration

- API key: `DASHSCOPE_API_KEY`
- Base URL: `DASHSCOPE_BASEURL`
- Default base URL: `https://dashscope.aliyuncs.com/compatible-mode/v1`
- Model used by the local server: `qwen-vl-plus`
